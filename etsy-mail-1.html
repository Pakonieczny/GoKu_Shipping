v2.0 — Sales Mode foundations — Step 1 of a 3-step rollout

When ON, a cron pulls active listings from the Etsy API every 30 min into Firestore. The AI's search_shop_listings tool returns matches with prices, images, and listing URLs. When OFF, the tool returns "disabled" and the AI replies without referencing specific products. Required for v2.0 Step 2 (sales agent).

Catalog status: not yet synced

When ON, each new thread runs through Claude Haiku once (cached 24h) and gets a colored "intent" badge in the thread list. Cheap (~$0.0002/message), purely additive — does NOT change reply behavior in Step 1. Required for v2.0 Step 2 sales-lead auto-routing.

v2.0 — Sales Mode (Step 2) — sales agent + pricing engine + state machine

When ON, threads classified as sales_lead at ≥ 0.7 confidence (or threads with an existing SalesContext) route to etsyMailSalesAgent, which runs a compressed sales funnel: discovery/spec can jump straight to quote when enough fixed-price details exist, option-sheet/collateral requests are answered directly, and pricing still comes from the deterministic pricing engine — the AI cannot quote below the floor. Requires intent classifier ON.

When OFF, only threads with an existing SalesContext (mid-funnel) route to the agent. Fresh sales leads still get an intent badge but go to the standard customer-service draft path. Recommended OFF during pilot — flip ON once stage progression is proven on a few threads.

When ON, only non-pricing Discovery and Spec drafts that ask a single true blocker or provide non-price options may auto-send via the Chrome extension. Any quote, revision, close artifact, Needs Review handoff, or safety-vetoed draft stays queued for operator approval — money commitments never auto-send. Operator approval is still required during pilot — flip ON when comfortable with the agent's reply quality.

During pilot, restrict the agent to specific threads. Even threads with active SalesContext are bypassed if not in this list. Empty the list once the pilot is solid.

Option Sheets — pricing source of truth (browse + import JSON)

Sheets are stored in EtsyMail_OptionSheets/<family>. You can edit seeds/option_sheets_seed.json and run seeds/import_seeds.js from a terminal, or upload a JSON file directly from this modal (owner-only). The AI's resolveQuote tool reads from these docs at runtime; cache TTL is 60 seconds.

Import option sheet JSON
Two accepted shapes: single family (one sheet object with family + sections keys) — replaces just that family. Or multi-family (top-level keys are family names, like option_sheets_seed.json) — replaces every named family. Each top-level _meta key is ignored. Writes are atomic per import.
